Assignment 1: This project contains implementations and experiments for statistical pattern recognition methods, it includes:
Feature Extraction & Bayesian Classification
Minimum risk classification (Rayleigh distributions)
Dimensionality Reduction
Singular Value Decomposition (SVD)
Principal Component Analysis (PCA)
Linear Discriminant Analysis (LDA)
PCA vs LDA comparison
Face dataset reconstruction experiments

Assignment 2: This project implements core machine learning algorithms from scratch and with standard libraries, covering:
Logistic Regression (with and without regularization)
Maximum Likelihood (MLE) and MAP estimation
K-Means clustering and image compression
Feedforward neural networks implemented with NumPy
Convolutional and dense neural networks using TensorFlow/Keras (Fashion-MNIST)
